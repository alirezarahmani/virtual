Introduction:
===
This is base on [Api Platform](https://api-platform.com/). 

How To Start:
=== 
run following commands in order:
> make build
> 
> make start 
> 
> make composer-install
> 
> make create-db
> 
> make update-schema
> 
> make fixture-load
> 
> make tests

Api doc:
===
`0.0.0.0:8000/api` is where you can find api doc for this
